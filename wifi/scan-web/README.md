# ESP32 Wi-Fi Scan Web

Educational ESP32 example demonstrating on-demand Wi-Fi scanning
with HTML output via a built-in HTTP server.

## Features
- Wi-Fi STA mode (DHCP)
- On-demand Wi-Fi scan via HTTP request
- HTML output in browser
- Serial debug output

## Hardware
Tested on:
- ESP32 Dev Module (ESP32-D0WD-V3)

## Build environment
- Arduino IDE (ESP32 Arduino Core)
- PlatformIO (Arduino framework) should also work

## How to run
1. Open the sketch in Arduino IDE
2. Set Wi-Fi credentials
3. Upload to ESP32
4. Open `http://<ESP32_IP>/`
